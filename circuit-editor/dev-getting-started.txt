All the libraries used here are deployed via package.json so there is no need to install any of them
manually, you can do that via `npm install command`. However, we maintain here a list of libraries we 
use in circuit editor.

- Install Node.js JavaScript runtime for working on circuit editor (https://nodejs.org/en/). 

- Install Vue.js JavaScript UI framework for working on circuit editor (https://vuejs.org/). 

- Vue CLI:

npm install -g @vue/cli

- Install Vuex state management pattern + library for Vue.js for working on circuit editor (https://vuex.vuejs.org/). 

npm install vuex 

- Install Vue Material: frontend framework for Vue (https://vuematerial.io/): 

npm install vue-material 

- Install Bootstrap, a CSS framework for working on circuit editor (https://getbootstrap.com/): 

npm install bootstrap 

- Install Popper.js for dropdowns (and components based on dropdown), tooltips, and popovers (https://popper.js.org/): 

npm i @popperjs/core 

- Install BootstrapVue front-end CSS library for working on circuit editor (https://bootstrap-vue.js.org/): 

npm install bootstrap-vue 

- Install js-yaml javascript library for translating json to yaml for circuit.

npm install js-yaml 

- Install html-to-image, a library for generating images from html.

npm install html-to-image 

- Install https://jscharting.com/ javascript charts library

npm install jscharting

- Install vue package to work with jscharting javascript charts

npm i -D jscharting-vue

cargo install cargo-generate

- Install JQuery javascript library:

npm install jquery

- Install Mathjs: https://www.npmjs.com/package/mathjs

npm install mathjs

- Install worker loader module for webpack: https://www.npmjs.com/package/worker-loader

npm install worker-loader
